# SOS Audio Detection

פרויקט זיהוי קולות מצוקה (SOS) באמצעות בינה מלאכותית.

## מבנה הפרויקט
- `data/raw/` - קבצי WAV מקוריים מחולקים לתיקיות לפי קטגוריה
- `data/processed/` - ספקטרוגרמות ומטריצות מעובדות
- `notebooks/` - מחברות Jupyter לניסויים
- `src/` - קוד הפרויקט הרשמי

## התקנה
```bash
pip install -r requirements.txt
```
